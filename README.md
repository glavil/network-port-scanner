# Python Port Scanner

Scanner de portas TCP desenvolvido em Python utilizando sockets e multithreading.

## Funcionalidades

- Escaneamento de portas TCP
- Resolução de domínio para IP
- Identificação de serviços
- Coleta de banners HTTP
- Execução paralela com ThreadPoolExecutor
- Salvamento dos resultados em arquivo

## Tecnologias

- Python 3
- Socket
- Threading
- Concurrent Futures

## Como executar

```bash
python scanner.py
```

## Exemplo

Digite o IP ou domínio alvo:
google.com

Digite a porta inicial:
1

Digite a porta final:
1024
